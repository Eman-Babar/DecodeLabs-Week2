# TechPulse – Modern SaaS Landing Page

**Project Overview**

This project was developed as part of my **Week 2 Internship Task at DecodeLabs**. TechPulse is a modern and responsive SaaS landing page designed to showcase innovative digital solutions through an engaging user experience and visually appealing interface.

**Features**

* Responsive Navigation Bar
* Interactive Mobile Menu
* Modern Hero Section
* Features Section using CSS Grid
* Services Section using Flexbox
* Team Showcase
* Call-to-Action (CTA) Section
* Professional Footer Layout
* Smooth and User-Friendly Design
* Mobile-Friendly Interface

**Technologies Used**

* HTML5
* CSS3
* JavaScript
* Google Fonts (Syne & DM Sans)

**Project Structure**

* `index.html` – Website structure and content
* `style.css` – Styling and responsive layouts
* `script.js` – Interactive functionality
* `README.md` – Project documentation

**Learning Outcomes**

Through this project, I learned:

* Creating advanced responsive layouts using CSS Grid and Flexbox
* Building interactive navigation menus with JavaScript
* Implementing modern UI/UX design principles
* Structuring scalable frontend projects
* Improving accessibility using semantic HTML
* Managing projects using Git and GitHub
